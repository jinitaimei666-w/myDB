## 简介

mall学习教程，架构、业务、技术要点全方位解析。mall项目（60k+star）是一套电商系统，使用现阶段主流技术实现。涵盖了SpringBoot、MyBatis、Elasticsearch、RabbitMQ、Redis、MongoDB、MySQL等技术，采用Docker容器化部署。

## 技术栈

本项目是一套涵盖主流Java技术栈的学习教程。涉及的主要技术包括：

- **后端框架**：SpringBoot 2.7.5
- **数据库**：MySQL 8.0
- **ORM框架**：MyBatis 3.5.10
- **缓存**：Redis
- **搜索引擎**：Elasticsearch
- **消息队列**：RabbitMQ
- **文档数据库**：MongoDB
- **容器化**：Docker
- **API文档**：Swagger UI

## 快速开始

### 环境要求

- JDK 1.8+
- Maven 3.3+
- MySQL 8.0+
- Redis

### 项目结构

该项目包含多个learning示例模块：

- `mall-tiny-01`：基础SpringBoot项目
- `mall-tiny-02`：整合Swagger
- `mall-tiny-03`：整合Redis
- 更多模块...
